# Stage Docs Index

문서 제목과 본문에서는 단계 이름을 직접 쓰고, 폴더 경로만 구조 구분용으로 사용한다.

## 문서
- `foundation-ontology.md` — 현재 foundation ontology 기준
- `foundation-engine.md` — 현재 foundation engine 기준
- `foundation-scenarios.md` — 현재 foundation scenario skeleton

## 원칙
- ontology 개념/관계 자체는 `../ontology/`가 소유한다.
- 이 폴더는 단계별 active 문서를 모으는 구조용 폴더다.
- roadmap와 checklist는 각각 `../roadmap/`에서 계속 소유한다.
